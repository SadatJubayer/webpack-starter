# webpack-starter
WebPack starter template with Babel, CSS, SASS, HTML and WebPackDevServer

> ### Install
``` 
    npm install
```
> ### Build webpack
``` 
    npm run build
```
> ### WebPack Dev Server

``` 
    npm start
```
